NPM

When building with ColonJs, it is recommended that you use the NPM installation method. It works well with module bundlers like Webpack.

$ npm install colonjs

Direct <script> Include

Download ColonJs and include with a script tag. Colon will be registered as a global variable. Get latest version here.

CDN:

Use the following code with the latest version for prototyping or for learning purposes:

<script src="https://cdn.jsdelivr.net/npm/colonjs"></script>

During production, link to and build off of a specific version number to avoid breakage from more recent versions:

<script src="https://cdn.jsdelivr.net/npm/colonjs@0.0.14"></script>

For native ES Modules, use the following compatible build:

<script type="module">
	import Colon from 'https://cdn.jsdelivr.net/npm/colonjs/dist/colon.esm.min.js'  
</script>

As an alternative to Jsdelivr CDN ColonJs is also available on unpkg CDN.
